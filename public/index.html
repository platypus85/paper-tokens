Paper Tokens Generator

Paper Tokens Generator is a free tool to easily generate and print round creature tokens to be used with RPGs or Tabletop games.

Just insert links to the creatures and change settings like name, size, quantity, count start ;) Once you press the button print you can save the token to PDF or send it directly to the printer!

Paper Tokens Generator is a work in progress tool, so come back for new improvements and updates and if you like this free tool and want to offer me a coffee press donate ♥️
